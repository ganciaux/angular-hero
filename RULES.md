# 🎮 Rules — Learning System

## 🧩 Missions

- 1 mission = 1 Angular concept
- Duration: 30–60 minutes
- 1 mission = 1 git commit

## 📈 Progression Rule

- Do not start the next mission before the current one is listed in Completed Missions in PROGRESS.md

## 🧠 Learning Rules

- No copy/paste (except project structure & setup)
- Try first, then ask for hints
- Understand before moving on

## 🔍 Code Review Rules

Each review includes:

1. What is correct
2. What can be improved
3. Mistakes
4. Angular best practices
5. Suggested solution (last)

## 🚫 Forbidden

- Using `any`
- Mixing concerns (UI / logic)
- Ignoring Angular style guide

## ✅ Mandatory

- Standalone components
- Signals for local state
- Clean architecture
